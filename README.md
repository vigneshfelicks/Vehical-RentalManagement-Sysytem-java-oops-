# Vehicle Rental Management System

## Overview

A Java OOP-based console application for managing vehicle rentals.

The system allows customers to rent and return vehicles while maintaining vehicle availability.

## Features

* Vehicle Registration
* Vehicle Availability Check
* Vehicle Rental
* Vehicle Return
* Customer Management
* Rental Tracking

## Concepts Used

* Object Oriented Programming (OOP)
* Encapsulation
* Inheritance
* Polymorphism
* Collections
* Exception Handling

## Technologies Used

* Java
* OOP Concepts
* Collections Framework

## Modules

* Vehicle Management
* Customer Management
* Rental Management

## Learning Outcomes

* Strong understanding of Java OOP principles
* Collection Framework usage
* Real-world business logic implementation
